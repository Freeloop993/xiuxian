# Xianxia Rules Engine v0.7

## 新增：挂机状态锁

规则：
1. 用户发送 `开始挂机X小时/分钟` 进入挂机状态锁。
2. 挂机期间禁止推进交互剧情与战斗探索。
3. 挂机期间只允许以下指令：
   - `挂机状态` / `剩余时间`
   - `领取挂机`
   - `结束挂机`
4. 挂机到时后，必须先 `领取挂机` 或 `结束挂机` 才能恢复普通交互。

## 挂机收益（当前版本）
- 按挂机时长结算真元与灵石（并触发小境界自动推进）。
- 提前 `结束挂机` 不结算挂机收益。
