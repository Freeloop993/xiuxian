# 丹药系统与立绘系统（轻量规则）

## 一、丹药系统（MVP）

说明：丹药数值与副作用由后端规则引擎结算；叙事文本继续由 OpenClaw 内置 LLM 增强，不接入独立第三方 LLM。

### 当前可用指令（已实现）
- 炼制纳气丹 / 炼制回春丹 / 炼制凝神丹
- 服用纳气丹 / 服用回春丹 / 服用凝神丹
- 选择男修 / 选择女修 / 选择自定义立绘

### 品质与副作用（已实现）
- 炼丹产出随机品质：凡品、良品、上品
- 品质影响：服丹收益倍率、丹毒增长幅度
- 丹毒 `>=100` 触发反噬：HP/MP 受损，凝神加成清零，丹毒回落到 60
- 闭关会缓慢化解丹毒

### 1) 丹药种类
- 纳气丹：提升真元（修为）
- 回春丹：恢复HP
- 凝神丹：给闭关提供短回合加成

### 2) 指令
- 炼制纳气丹（消耗30灵石）
- 炼制回春丹（消耗40灵石）
- 炼制凝神丹（消耗60灵石）
- 服用纳气丹
- 服用回春丹
- 服用凝神丹

### 3) 规则
- 服用纳气丹：增加真元，随后按原规则自动尝试小境界推进。
- 服用回春丹：恢复HP上限内。
- 服用凝神丹：获得`focusBuffTurns`，闭关时额外修为收益，按回合递减。

### 4) LLM职责
- 后端只负责扣资源、加数值、状态推进。
- LLM负责把结果叙事化（炼丹过程、丹毒风险、药香描写、人物心境）。
